## About The Project
This is a bookstore back-end api. It's an app for managing bookstore database.
It's built using Ruby on Rails.

### Built With
This project was built using these technologies.
* Ruby
* Ruby on Rails
* Rack-Cors gem
* Hirb gem
* VScode
